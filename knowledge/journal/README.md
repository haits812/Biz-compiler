# Journal

このディレクトリは、作業セッションのログ、検討過程、まだ整理していないメモを置く場所である。

## 使い方

- 1セッションまたは1テーマごとに `YYYY-MM-DD-<topic>.md` を作る。
- ここには長い思考過程、試行錯誤、比較検討、未確定メモを置いてよい。
- 後から再利用できる決定は `knowledge/docs/decisions/` へ昇格する。
- 常時参照すべき短い事実だけ `MEMORY.md` へ昇格する。

## 書かないもの

- 永続的な作業規約。これは `AGENTS.md`。
- 判断prior。これは `COMPASS.md`。
- 要求定義本文。これは `knowledge/docs/requirements/`。
