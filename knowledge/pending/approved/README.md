# Approved Pending Updates

このディレクトリは、人間が「正式反映してよい」と確認したpending候補を置く。

`approved` はまだ正式ファイルへ反映済みという意味ではない。正式反映後は `knowledge/ops/skills/knowledge-curation/pending-review.ps1 applied` で `knowledge/journal/pending-applied/` へ移す。

## サブディレクトリ

- `memory/`: `MEMORY.md` へ反映してよい候補。
- `compass/`: `COMPASS.md` へ反映してよい候補。
- `decisions/`: `knowledge/docs/decisions/` へ昇格してよい候補。
